# TP1 - Pilas en C

Trabajo práctico desarrollado en C para la materia Programación I (Tecnicatura en Programación - UTN).

El programa implementa una estructura de datos tipo **Pila** y permite realizar distintas operaciones mediante un menú interactivo por consola.

## Funcionalidades implementadas

El programa permite:

1. Cargar una pila con valores
2. Pasar elementos de una pila A a una pila B
3. Pasar elementos de una pila A a una pila B manteniendo el orden
4. Encontrar el menor elemento y eliminarlo de la pila
5. Ordenar una pila mediante ordenamiento por selección
6. Insertar un número en una pila ordenada
7. Ordenar una pila mediante ordenamiento por inserción

## Conceptos aplicados

- Estructuras (struct)
- Modularización
- Pasaje por referencia
- Manipulación de pilas
- Algoritmos de ordenamiento
- Búsqueda de mínimo
- Funciones auxiliares
- Manejo de memoria
