# TP2 - Arrays, Matrices y Strings en C

Trabajo práctico desarrollado en C para la materia Programación I  
Tecnicatura en Programación - UTN

Este proyecto implementa funciones para el manejo de:

- Matrices de enteros
- Cálculos sobre matrices
- Búsqueda de elementos
- Arreglos de strings
- Ordenamiento alfabético
- Búsqueda de palabras

---

##  Funcionalidades implementadas

###  Matrices de enteros

- Carga manual de una matriz
- Carga automática con números aleatorios
- Mostrar matriz en formato matricial
- Suma total de los elementos
- Cálculo de promedio
- Búsqueda de un número dentro de la matriz

---

###  Arreglo de palabras (strings)

- Carga dinámica de palabras
- Mostrar palabras cargadas
- Búsqueda secuencial de una palabra
- Ordenamiento alfabético (método por selección)
- Eliminación del menor elemento durante el proceso de ordenamiento

---

##  Conceptos aplicados

- Arrays unidimensionales y bidimensionales
- Strings en C
- Uso de `strcmp`, `strcpy`
- Punteros como parámetros
- Pasaje por referencia
- Modularización
- Algoritmos de búsqueda
- Ordenamiento por selección adaptado a strings
- Uso de `rand()` y `srand()`
