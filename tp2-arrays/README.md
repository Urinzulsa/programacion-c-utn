# TP3 - Arreglos y Funciones en C

Trabajo práctico desarrollado en C para la materia Programación I  
Tecnicatura en Programación - UTN

Este proyecto implementa múltiples ejercicios enfocados en el manejo de arreglos de distintos tipos de datos (int, float, char), uso de funciones, punteros y estructuras como Pilas.

El programa funciona mediante un menú interactivo por consola que permite ejecutar cada ejercicio.

---

##  Funcionalidades implementadas

###  Arreglos de enteros

- Carga de arreglo y retorno de cantidad de elementos válidos
- Mostrar arreglo por pantalla
- Suma de elementos
- Pasaje de arreglo a estructura Pila

---

###  Arreglos de float

- Carga de arreglo de números reales
- Suma de elementos del arreglo

---

###  Arreglos de caracteres

- Carga dinámica de caracteres
- Búsqueda de un carácter dentro del arreglo
- Inserción de un carácter en arreglo ordenado
- Obtención del carácter máximo
- Verificación de arreglo capicúa (palíndromo)

---

##  Conceptos aplicados

- Arrays unidimensionales
- Manejo de válidos
- Modularización
- Pasaje por valor y por referencia
- Punteros
- Uso de estructuras (Pila)
- Algoritmos básicos de búsqueda
- Inserción en arreglo ordenado
- Comparación de elementos
- Lógica de palíndromos

El objetivo del trabajo es reforzar:
