# Programación en C – Tecnicatura UTN

Repositorio con trabajos prácticos desarrollados en C durante las materias Programación I de la Tecnicatura en Programación (UTN).

##  Contenidos trabajados

Durante la cursada se desarrollaron ejercicios y trabajos prácticos aplicando los siguientes conceptos:

- Arrays y matrices
- Arreglos dinámicos
- Estructuras (structs)
- Arreglos paralelos
- Pilas
- Archivos (lectura y escritura)
- Punteros
- Punteros dobles
- Modularización del código
- Manejo de memoria dinámica (malloc, realloc, free)

##  Objetivos académicos

- Desarrollar lógica de programación estructurada.
- Comprender el manejo de memoria en C.
- Implementar estructuras de datos básicas.
- Trabajar con persistencia de datos mediante archivos.
- Aplicar buenas prácticas de organización y modularización.

##  Tecnologías utilizadas

- Lenguaje: C
- Compilador: GCC

##  Cómo compilar y ejecutar

Compilar:
gcc main.c -o programa

Ejecutar:
./programa

##  Aprendizajes

Estos trabajos permitieron afianzar el pensamiento lógico, la comprensión del manejo de memoria dinámica y la importancia de la correcta gestión de recursos en programación de bajo nivel.
